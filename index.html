Semantic vs. Non-Semantic HTML

How to Write "Good" Code

Semantic Non-Semantic Explanation
<header> <div class="header"> Use the header tag to encase the content above the <body>.
<nav> <div class="nav"> <nav> Houses the navigational links. This can be found in the <header> and <footer>.
<main> <div> Main content of the document
<footer> <div class="footer"> Found at the bottom of the body. May contain copyright info or a site directory.
<section> <div> Defines elements in a document that share the same theme.
<article> <div> Holds info that makes sense For example, articles or blog posts.
<aside> <div> Additional info that can enhance, but not required.
<video> Adds videos. Needs a closing tag.
<audio> Implements audio. Needs a closing tag.
<embed> Implements any type of media. Does not need a closing tag.
**The above three mentioned all use the src attribute to link to the source of the content.
<figure> Encapsulates media such as an image or diagram.
<figcaption> Used to describe the <figure>
